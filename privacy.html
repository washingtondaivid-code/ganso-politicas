INTRODUÇÃO Este aplicativo respeita a privacidade dos seus usuários e descreve como coletamos, usamos e protegemos seus dados. DADOS COLETADOS Podemos coletar os seguintes dados para funcionamento do aplicativo: - Nome do usuário - Endereço de e-mail - Dados de autenticação (login Google) - ID de usuário - Dados de uso dentro do aplicativo - Arquivos ou mídias enviados pelo usuário (quando aplicável) --- 3. FINALIDADE DO USO DOS DADOS Os dados coletados são utilizados para: - Permitir autenticação e login de usuários - Personalizar a experiência do usuário - Armazenar dados necessários ao funcionamento do aplicativo - Salvar conteúdos enviados pelo usuário - Melhorar funcionalidades e desempenho do aplicativo --- 4. ARMAZENAMENTO E SEGURANÇA Os dados são armazenados de forma segura em serviços de terceiros confiáveis. --- 5. SERVIÇOS DE TERCEIROS Utilizamos serviços de terceiros que podem coletar e processar dados necessários para o funcionamento do aplicativo: --- Supabase Utilizado para autenticação, banco de dados e backend do aplicativo. Dados processados: - E-mail do usuário - ID de autenticação - Dados de uso do aplicativo https://supabase.com --- Backblaze B2 Utilizado para armazenamento de arquivos e mídias enviadas pelos usuários (como músicas e conteúdos). Dados processados: - Arquivos enviados - Metadados de arquivos - Nome dos arquivos https://www.backblaze.com/b2/cloud-storage.html --- Google Utilizado para autenticação via Google Sign-In. Dados processados: - Nome - E-mail - ID da conta Google https://policies.google.com/privacy --- 6. COMPARTILHAMENTO DE DADOS Não vendemos, alugamos ou comercializamos dados pessoais.
