Política de Privacidade

Esta Política de Privacidade descreve como os aplicativos da Ganso Produções coletam, utilizam e protegem os dados dos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

1. Apps cobertos por esta política

  • MuralChat
  • Aplicativos de Música
  • Sistema PDV (Ponto de Venda / Caixa)
  • Ganso Studio

2. Dados coletados

Podemos coletar os seguintes dados:

  • Nome e e-mail (via login ou autenticação Google)
  • Dados de autenticação
  • Dados de uso do aplicativo
  • Mensagens e interações (no caso de chat)
  • Arquivos enviados ou armazenados (músicas, mídias, documentos)
  • Dados de vendas e clientes (apps PDV)

3. Finalidade do uso dos dados

  • Autenticação de usuários
  • Funcionamento dos aplicativos
  • Armazenamento de informações
  • Processamento de dados internos
  • Melhoria da experiência do usuário

4. Categorias por tipo de aplicativo

🎵 Aplicativos de Música

Utilizam armazenamento de arquivos de mídia (como músicas e áudios) através do serviço Backblaze B2.

🧾 Aplicativos PDV (Caixa / Loja)

Utilizam Supabase para autenticação, banco de dados e gerenciamento de informações de vendas e clientes.

💬 MuralChat

Utiliza autenticação de usuários e armazenamento de mensagens e interações sociais.

🎛 Ganso Studio

Pode utilizar autenticação, armazenamento e recursos internos conforme funcionalidades do aplicativo.

5. Serviços de terceiros

Supabase

Utilizado para banco de dados, autenticação e serviços backend.

https://supabase.com

Backblaze B2

Utilizado para armazenamento de arquivos e mídias (como músicas).

https://www.backblaze.com/b2/cloud-storage.html

Google

Utilizado para autenticação e login de usuários.

https://policies.google.com/privacy

6. Armazenamento de dados

Os dados são armazenados em serviços seguros de terceiros como Supabase e Backblaze B2. Os dados são mantidos apenas pelo tempo necessário para o funcionamento dos aplicativos.

7. Compartilhamento de dados

Não vendemos dados pessoais. Alguns dados podem ser processados por serviços de terceiros apenas para funcionamento dos aplicativos.

8. Segurança

Utilizamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, alteração ou perda.

9. Direitos do usuário (LGPD)

O usuário pode solicitar a qualquer momento:

  • Acesso aos seus dados
  • Correção de dados incorretos
  • Exclusão de seus dados
  • Revogação de consentimento

10. Exclusão de dados

O usuário pode solicitar a exclusão de seus dados entrando em contato pelo e-mail do desenvolvedor. A solicitação será atendida em prazo razoável.
