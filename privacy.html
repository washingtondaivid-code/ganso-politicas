INTRODUÇÃO Esta Política de Privacidade descreve como coletamos, utilizamos, armazenamos e protegemos as informações dos usuários dos aplicativos desenvolvidos pela Ganso Produções. Ao utilizar este aplicativo, você concorda com as práticas descritas nesta política. --- # 2. DADOS COLETADOS Dependendo das funcionalidades utilizadas, poderemos coletar: * Nome do usuário; * Endereço de e-mail; * Dados de autenticação (Google Sign-In ou outros provedores); * Identificador único do usuário (User ID); * Dados de uso e interação com o aplicativo; * Arquivos, músicas, imagens, vídeos ou outros conteúdos enviados pelo usuário; * Metadados associados aos arquivos processados. --- # 3. FINALIDADE DO USO DOS DADOS Os dados coletados podem ser utilizados para: * Autenticação e gerenciamento de contas; * Funcionamento das funcionalidades do aplicativo; * Processamento de músicas, áudios e mídias; * Geração de conteúdos por inteligência artificial; * Armazenamento seguro de arquivos do usuário; * Personalização da experiência do usuário; * Melhorias de desempenho, estabilidade e segurança; * Suporte técnico e atendimento ao usuário. --- # 4. ARMAZENAMENTO E SEGURANÇA Os dados são armazenados utilizando provedores de infraestrutura reconhecidos internacionalmente e protegidos por mecanismos de autenticação, controle de acesso e criptografia compatíveis com os padrões do mercado. Adotamos medidas razoáveis para evitar acessos não autorizados, alterações indevidas, divulgação ou destruição dos dados pessoais. --- # 5. SERVIÇOS DE TERCEIROS Para fornecer determinadas funcionalidades, utilizamos os seguintes serviços: --- Supabase Utilizado como plataforma de backend, autenticação, banco de dados e gerenciamento das informações dos usuários. Dados processados: * E-mail; * Identificador do usuário; * Dados de autenticação; * Dados necessários ao funcionamento do aplicativo. https://supabase.com --- Backblaze B2 Utilizado para armazenamento seguro de músicas, imagens, vídeos e demais arquivos enviados pelos usuários. Dados processados: * Arquivos enviados; * Metadados dos arquivos; * Informações técnicas relacionadas ao armazenamento. https://www.backblaze.com/b2/cloud-storage.html --- OpenAI Utilizado para funcionalidades de inteligência artificial, geração de conteúdo, assistentes virtuais, processamento de linguagem natural e recursos inteligentes disponibilizados pelo aplicativo. Dependendo da funcionalidade utilizada, textos, comandos ou conteúdos enviados pelo usuário poderão ser processados para execução das tarefas solicitadas. Dados processados: * Textos enviados pelo usuário; * Solicitações e comandos realizados; * Dados estritamente necessários para a execução das funcionalidades de IA. https://openai.com/privacy --- Replicate Utilizado para execução de modelos avançados de inteligência artificial voltados ao processamento de áudio, música, imagens e outras funcionalidades especializadas. Dados processados: * Arquivos enviados para processamento; * Dados necessários para execução dos modelos utilizados. https://replicate.com/privacy --- Daily.co Utilizado para recursos de comunicação em tempo real, incluindo chamadas de áudio, vídeo, streaming e sessões colaborativas quando disponíveis no aplicativo. Dados processados: * Identificadores de sessão; * Informações técnicas de conexão; * Dados necessários para transmissão de áudio e vídeo. https://www.daily.co/legal/privacy --- Demucs Utilizado para processamento musical e separação de faixas de áudio, permitindo funcionalidades como extração de voz, instrumentos e edição musical. Os arquivos processados são utilizados exclusivamente para executar as solicitações realizadas pelo próprio usuário. Dados processados: * Arquivos de áudio enviados; * Metadados técnicos relacionados ao processamento. https://github.com/facebookresearch/demucs --- Google Utilizado para autenticação por meio do Google Sign-In e demais integrações necessárias à plataforma Android. Dados processados: * Nome do usuário; * Endereço de e-mail; * Identificador da conta Google. https://policies.google.com/privacy --- # 6.
